# Logo Component Preview

Ce dossier contient les previews visuelles du composant Logo.

## Génération de la Preview

Pour générer une preview du composant Logo :

1. Démarrer l'application Laravel Sail :
   ```bash
   ./vendor/bin/sail up -d
   ```

2. Accéder à la page de preview :
   ```
   http://localhost/design-system/components/logo
   ```

3. Prendre un screenshot de la page complète et l'enregistrer comme `logo-preview.png` dans ce dossier.

## Contenu de la Preview

La page de preview affiche :
- Toutes les variantes de taille (sm, md, lg, xl)
- Options avec et sans scanlines
- Exemples d'usage dans différents contextes (landing page, navigation)
