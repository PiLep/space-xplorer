# Propositions de Règles Techniques

Ce dossier contient les propositions de nouvelles règles techniques proposées par Morgan (Architect) et Sam (Lead Developer).

## Format

Chaque proposition suit le format : `PROPOSAL-{YYYY-MM-DD}-{titre-kebab-case}.md`

## Processus

1. **Proposition** : Morgan ou Sam crée une proposition
2. **Review Interne** : L'autre agent technique review
3. **Validation Humaine** : ⚠️ Validation humaine requise avant application
4. **Application** : Une fois validée, la règle est ajoutée à TECHNICAL_RULES.md
