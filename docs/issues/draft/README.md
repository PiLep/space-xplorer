# Dossier Draft - Documents de Travail

Ce dossier contient les documents de travail temporaires créés pendant le processus d'élaboration d'issues.

## Usage

Quand l'agent Product (Alex) utilise l'action `elaborate-issue`, il peut créer un document de travail temporaire dans ce dossier pour suivre la progression de l'élaboration.

**Format du nom de fichier** : `DRAFT-{titre-kebab-case}.md`

**Exemple** : `DRAFT-systeme-decouverte-planetes.md`

## Nettoyage

Une fois l'issue finale créée, le document de travail (`DRAFT-*.md`) doit être supprimé car toutes les informations sont intégrées dans l'issue finale.

Ces documents sont temporaires et ne doivent pas être commités dans le repository principal. Ils servent uniquement de support pendant le processus d'élaboration.
